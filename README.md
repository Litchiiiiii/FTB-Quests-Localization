# 概述

这是FTB-Quests-Localization（未实现）的 fork，一个 FTBQuest 本地化 Mod。

# 如何使用

在聊天栏中输入 /ftblang export <originalLangCode> 指令，如 /ftblang export en_us；

等待屏幕中弹出导出路径提示，即表示导出成功；

# 后续操作

只需翻译语言文件即可完成任务翻译。
